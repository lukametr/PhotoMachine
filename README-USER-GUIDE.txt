============================
ფოტოჯიხურის აპლიკაციის მომხმარებლის გზამკვლევი
============================

1. აპლიკაციის დანიშნულება
-------------------------
ეს აპლიკაცია განკუთვნილია Windows პლანშეტისთვის, ფოტოჯიხურის რეჟიმში (Kiosk Mode).
მომხმარებელი გადახდის გარეშე ვერ გამოიყენებს სერვისს.
გადახდის შემდეგ გადამისამართდება ფოტოს გადაღების პროგრამაში (მაგ: dslrBooth).

2. აპლიკაციის ინსტალაცია და გაშვება
-----------------------------------
1. გახსენით მოწოდებული ინსტალატორი (მაგ: PhotoMachine-Setup.exe) ან გამშვები ფაილი (PhotoMachine.exe).
2. დააინსტალირეთ აპლიკაცია ეკრანზე მითითებული ინსტრუქციების მიხედვით (თუ არის setup).
3. აპლიკაცია ავტომატურად გაიხსნება ან გაუშვით PhotoMachine ხატულაზე დაჭერით.

აპლიკაცია გაიხსნება სრულ ეკრანზე და იმუშავებს მხოლოდ აპლიკაციის რეჟიმში.

3. მომხმარებლის ინტერფეისი
---------------------------
- მთავარი ეკრანი: "დასაწყებად შეეხეთ ეკრანს" (ან "Tap to Start").
- გადახდის ეკრანი: "გადახდის დასაწყებად დაადეთ ბარათი ან ტელეფონი".
- წარმატებული გადახდა: "გადახდა წარმატებულია".
- წარუმატებელი გადახდა: "გადახდა წარუმატებელია".
- Retry/Cancel ღილაკები წარუმატებლობისას.
- ენების შეცვლა შესაძლებელია ზედა მარჯვენა კუთხიდან (ქართული/English).

4. გადახდის პროცესი
-------------------
- მომხმარებელი აჭერს "დასაწყებად შეეხეთ ეკრანს".
- გადადის გადახდის ეკრანზე.
- გადახდის წარმატების შემთხვევაში გადადის ფოტოს გადაღების პროგრამაში.
- გადახდის პროცესი ამჟამად იმიტირებულია (სატესტო რეჟიმი). რეალური ტერმინალის ინტეგრაცია საჭიროებს სპეციალისტს.
- წარუმატებლობის შემთხვევაში Retry ან Cancel.

5. dslrBooth-ის გაშვება და აპის ქცევა
--------------------------------------
- გადახდის წარმატების შემდეგ აპლიკაცია ავტომატურად იმალება და იხსნება dslrBooth.
- dslrBooth-ის დახურვის შემდეგ აპლიკაცია ავტომატურად ბრუნდება საწყის ეკრანზე.
- მომხმარებელს არ აქვს წვდომა სხვა პროგრამებზე dslrBooth-ის გარდა.

6. ადმინისტრატორის გამოსვლა აპლიკაციიდან
-----------------------------------------
- ეკრანის ქვედა მარჯვენა კუთხეშია უხილავი (ფარული) ზონა.
- ამ ზონაზე დაჭერით იხსნება პინ კოდის ფანჯარა.
- სწორი პინით (3223) აპლიკაცია დაიხურება.
- არასწორი პინის შემთხვევაში ჩანს შეცდომის შეტყობინება.
- 3-ჯერ არასწორი პინის შემდეგ ითხოვს ადმინისტრატორის პაროლს (Order66).

7. dslrBooth Event API სერვერის აუცილებლობა
-----------------------
- PhotoMachine აპლიკაციის სწორად მუშაობისთვის აუცილებელია, რომ dslrBooth Event API (ან Event აპი) იყოს გაშვებული მთავარ ეკრანზე იმავე მოწყობილობაზე.
- თუ Event API არ მუშაობს ან არ არის გაშვებული, გადახდის შემდეგ ფოტოსესია არ დაიწყება.
- Event API-ს მისამართი და პარამეტრები იხილეთ ადმინისტრატორისგან ან ტექნიკური დოკუმენტაციიდან.
